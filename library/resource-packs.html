Installation Instructions

for Bedrock Packs:
  • Download the .mcpack file and navigate to it in your file browser
  • Open the file via a double click or tap and if prompted choose to open it with Minecraft
    (On android, if you don't get the prompt or don't have Minecraft as an option then download a third party app that can read it like FX File Explorer and try to open it from there)
  • Your Minecraft app should open and start importing the pack
  • After the import is finished restart your game and then apply the pack in your settings
for Java Packs:
  • Download the .zip file and navigate to it in your file browser
  • Open Minecraft Java Edition, open options, then open resource packs
  • Click the "Open Resource Pack Folder" button
  • Drag the .zip pack into the resource pack folder and you're done
  • Exit the Resource Pack menu and then open it again and your resource pack will be there
